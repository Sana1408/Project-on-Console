Console is the infrastructure for console applications in Microsoft Windows. An instance of a Windows Console has a screen buffer and an input buffer. It allows console apps to run inside a window or in hardware text mode (so as to occupy the entire screen). To open the browser console, right-click on the page and select Inspect. You should use the console. log() method to print to console JavaScript.
